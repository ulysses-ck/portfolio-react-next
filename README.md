## Get Started

Clone repo

```
git clone https://github.com/ulysses-ck/portfolio-react-next
```

Install dependencies

```sh
npm install
```

Start development server.

```bash
npm run dev
```

By default, the server listen at:
[http://localhost:3000](http://localhost:3000)

For linting, use:

```sh
npm run lint
```

And formatting, use:

```sh
npm run format
```
